Répertoire des releases Kornog
================================

Déposez ici vos fichiers de release à distribuer via le site :

  kornog-android-vX.X.X.apk        → APK Android
  kornog-windows-vX.X.X-setup.exe  → Installeur Windows
  kornog-windows-vX.X.X.zip        → Archive Windows portable
  kornog-linux-vX.X.X.AppImage     → AppImage Linux
  kornog-linux-vX.X.X.deb          → Paquet Debian/Ubuntu

Une fois le fichier déposé, mettez à jour le href correspondant dans index.html :

  <a href="releases/kornog-android-vX.X.X.apk" ...>Télécharger l'APK</a>

Et changez le badge de statut :
  status--soon  → bientôt disponible
  status--beta  → version bêta
  status--available → disponible

N'oubliez pas de commiter et pusher après chaque ajout.
